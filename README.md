# Technische Mechanik mit SMath Studio

Dieses Repository enthält die SMath-Rechenblätter des Buchs "Technische Mechanik mit SMath Studio" von Martin Kraska.

Die Aufgabenstellungen sind im Wesentlichen aus Henning, G., Jahr, A. und Mrowka, U.: Technische Mechanik mit Mathcad, Matlab und Maple (Vieweg, 2004) 
übernommen.
Der damit mögliche Vergleich zu Mathcad ist insbesondere für Umsteiger auf SMath Studio interessant.
